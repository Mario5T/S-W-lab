My Projects

Manhwa Review Website

The Manhwa Review Website is an ongoing web development project aimed at creating a comprehensive platform for manhwa enthusiasts. The platform will allow users to explore, review, and discuss their favorite Korean comics (manhwa) in a user-friendly and engaging environment. Key Features: Manhwa Catalog: A well-organized database featuring popular and lesser-known manhwa titles, categorized by genres, release dates, and ratings. User Reviews & Ratings: Registered users can post detailed reviews and rate manhwa titles, contributing to a vibrant community-driven rating system. Discussion Forums: Dedicated spaces for fans to discuss plot twists, character development, and fan theories. Recommendations Engine: Personalized suggestions based on users' reading history, ratings, and favorite genres. Search & Filter Options: Advanced search capabilities to help users find manhwa titles based on keywords, authors, or themes. News & Updates: A section to keep readers informed about new releases, adaptations, and industry news. Technologies Involved: Frontend: Built with modern frameworks like React or Angular for a responsive and interactive user experience. Backend: Powered by Node.js or Django, ensuring fast and secure handling of user data and interactions. Database: A robust database system like MongoDB or PostgreSQL to store and retrieve manhwa information efficiently. UI/UX Design: Aesthetic and intuitive design tailored for manhwa fans, ensuring ease of navigation and engagement. Current Progress: Phase 1: Database structure and backend architecture are under development. Phase 2: UI/UX prototypes have been designed, focusing on mobile and desktop responsiveness. Phase 3: Integration of the review and rating system is underway, with beta testing scheduled for early completion. This project aims to become a go-to platform for manhwa fans, fostering a global community while promoting the art and storytelling of Korean comics.

Online Library for Novels

The Online Library for Novels is an ongoing project designed to create a digital platform where users can access a wide variety of novels across different genres. The platform aims to provide a seamless and enjoyable reading experience for book enthusiasts while offering tools to discover, organize, and discuss their favorite books. Key Features: Extensive Novel Collection: A growing library of novels spanning genres like romance, mystery, fantasy, science fiction, historical fiction, and more. Genre-Specific Browsing: Users can explore novels sorted by genres, themes, and popularity, making it easy to find books tailored to their interests. User Accounts: Personalized accounts that allow users to save favorite books, track reading progress, and create custom reading lists. Advanced Search & Filtering: Robust search functionality to find books by title, author, genre, or keywords, coupled with filters like ratings and publication date. Community Reviews & Ratings: A space for readers to leave reviews, rate books, and engage in discussions about their favorite titles. E-Reading Functionality: Integrated online readers with adjustable settings (font size, themes, and night mode) to enhance the reading experience. Recommendation System: AI-driven suggestions based on users’ reading history, preferences, and ratings. Weekly Highlights & New Arrivals: Regularly updated sections showcasing trending novels, editor’s picks, and newly added titles. Technologies Involved: Frontend: Responsive design using frameworks like React.js or Vue.js for an intuitive user interface. Backend: Developed with robust technologies like Node.js or Django for secure and scalable user and book management. Database: SQL or NoSQL databases (e.g., MySQL, MongoDB) for storing book details, user data, and reviews. E-Reader Integration: Tools like EPUB.js for seamless reading directly on the platform. Current Progress: Phase 1: Backend and database architecture are nearly complete, focusing on scalability and search optimization. Phase 2: Frontend design is in progress, with prototypes created for desktop and mobile interfaces. Phase 3: Integration of the reading system and user profile features is underway, with initial testing scheduled for completion soon. This online library aims to become a go-to destination for novel enthusiasts, promoting a love for reading while fostering a dynamic literary community.
